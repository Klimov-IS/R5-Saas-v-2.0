## Цель

Сформировать единый Source of Truth по коммуникации в чатах маркетплейсов:
- единая логика общения
- стандарты шаблонов
- правила компенсаций
- структура ID заявок
- алгоритм рассылки
- источники правил
- логика выбора артикула

Документ должен стать:
- операционным стандартом
- базой для автоматизации
- основой для будущего AI-агента общения

## Правки от владельца (2026-02-28)

1. **Компенсация:** Фиксированная по артикулу, НЕ зависит от рейтинга (1-2-3★ — одна сумма). Строго по правилам артикула.
2. **Удаление:** Упоминать можно, но НЕ в начале диалога. Стратегия: узнать проблему → обсудить → "согласовали бонус" → "с вас удаление, с нас кешбэк"
3. **Телефон R5:** +79950509459 (актуальный). В будущем — настраиваемый в ПО.
4. **4★:** Только планируется, пока не работает.
5. **`&tab=unread`:** Баг, нужно убрать.
6. **Макс компенсация:** До 2000₽ на практике, ограничений нет.
7. **Полный возврат:** Некоторые магазины предлагают полный возврат через WB.
8. **OZON:** Только подключаем. Шаблоны WB и OZON раздельные.
9. **600₽:** Финансовая составляющая R5, НЕ относится к чатам. Убрать из промпта.
10. **Рассылка:** Планируется новая — 30 дней × раз в 3 дня.
11. **Подтверждение удаления:** Клиент пишет → API к WB → проверка → TG → сообщение с ID заявки.
