## Goal

UI для биллинга в кабинете: настройка цены, просмотр счетов, учёт оплат, виджет задолженности.

## Scope

### Компоненты

**BillingSettingsCard:**
- Цена за удаление (editable number)
- Способ расчёта (monthly/weekly)
- Email для счетов

**BillingSummaryCard:**
- Крупно: цена за удаление
- Удалено за период: N
- Начислено: сумма
- Оплачено: сумма
- Задолженность: выделена цветом (зелёный если 0, жёлтый/красный если долг)

**InvoicesTable:**
- Таблица счетов: период | удалений | сумма | статус | действия
- Кнопка "Создать счёт" → модал с выбором периода
- Статусы: draft (серый) → sent (синий) → paid (зелёный) / overdue (красный)

**PaymentsLog:**
- Таблица оплат: дата | сумма | метод | привязка к счёту
- Кнопка "Записать оплату"
